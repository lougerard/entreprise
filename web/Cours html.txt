<h2>Qu'est-ce que HTML?</h2>
<p>Le HTML est un standard de balisage pour la création de pages web</p>
<ul>
<li>HTML signifie "Hyper Text Markup Language"</li>
  <li>HTML décrit la structure des pages web utilisant du balisage</li>
  <li>Les pages HTML sont construites à partir de blocks HTML</li>
  <li>Les éléments HTML sont représentés par des tags</li>
  <li>HTML tags label pieces of content such as "heading", "paragraph", "table", and so on</li>
  <li>Browsers do not display the HTML tags, but use them to render the content of the page</li>
</ul>

<h2><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Balises HTML</font></font></h2>
<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Les balises HTML sont des noms d’éléments entourés de crochets:</font></font></p>
<ul>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Les balises HTML viennent normalement </font></font><b><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">par paires</font></font></b><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> comme </font></font><code class="w3-codespan">&lt;p&gt;</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">et</font></font><code class="w3-codespan">&lt;/p&gt;</code></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">La première balise dans une paire est la </font></font><b><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">balise de début,</font></font></b><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> la deuxième balise est la </font></font><b><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">balise de fin</font></font></b></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">La balise de fin est écrite comme la balise de début, mais avec une </font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">barre oblique</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">  
insérée avant le nom de la balise</font></font></li>
</ul>
